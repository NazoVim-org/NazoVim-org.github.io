Features

Everything you need. Nothing you don't.

⚡
Fast Startup
lazy.nvim によるアグレッシブな遅延ロード。起動を邪魔するものは VeryLazy 以降に追いやる。
🔧
Full LSP
TypeScript / Lua / Ruby / Nix / C / Java / Zig など多言語対応。mason + nixd の両刀構成。
🎨
Kanagawa
kanagawa-dragon テーマ + 透明背景。ターミナルと完全に溶け込む洗練された見た目。
🧠
AI Integration
Claude Code と opencode を両方サポート。claudecode.nvim + opencode.nvim。
🌊
snacks.nvim
Dashboard・Picker・Zen mode・セッション管理をひとつのプラグインで統一。
📦
Nix Ready
nix run で隔離起動。nix develop で LSP 完備の devShell。既存設定を汚さない。
🐛
DAP Built-in
JavaScript / TypeScript のデバッグ環境を最小構成で内包。nvim-dap + vscode-js-debug。
🔍
Fuzzy Finding
snacks.nvim Picker (ivy layout) をメイン、Telescope をサブとして併用。快適な検索体験。

Installation

Pick your method. Start editing in minutes.

Method 1 — nix run
RECOMMENDED

設定を ~/.config/nvim-nazozokc に隔離展開。既存の Neovim 設定を一切汚さない。

❯ nix run github:nazozokc/nazozokc.nvim.config
Method 2 — nix develop

LSP とフォーマッターが揃った devShell に入る。NVIM_APPNAME が自動でセットされる。

❯ nix develop github:nazozokc/nazozokc.nvim.config nvim
Method 3 — git clone
OVERWRITES ~/.config/nvim

Nix 不要。初回起動時に lazy.nvim が全プラグインを自動インストール。バックアップ推奨。

❯ git clone https://github.com/nazozokc/ nazozokc.nvim.config.git ~/.config/nvim nvim
